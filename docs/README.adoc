== How this index is maintained

Authoritative index. Whenever a new document is added (or relocated),
update this file at the same time. The pattern PR is: (1) the doc
change, (2) one-line addition here, (3) update of any references in
the root link:../README.adoc[README] if the new doc warrants top-level
visibility.

For a navigation taxonomy reference, see `rsr-template-repo/docs/README.adoc`.
