=== Working Features Today

* Logtalk rules engine with formal schema (`rule_schema.lgt`)
* Bot integration protocol (`bot_integration.lgt`)
* ArangoDB graph model with 9 collections, 16 edges, 3 graphs
* 20+ AQL query templates
* Gitbot fleet shared context layer
* Robot-repo-automaton Rust executor
* Glambot/Finishing-bot with fleet integration
* Test coverage across all components

== Risk Assessment

=== Technical Risks

[cols="2,1,2"]
|===
| Risk | Severity | Mitigation

| Liquid Haskell complexity
| Medium
| Start with QuickCheck, Liquid optional

| Learning pipeline accuracy
| High
| Human review gate, high threshold

| Multi-forge API changes
| Medium
| Abstract adapter interface, tests

| Dragonfly scaling
| Low
| Horizontal scaling built-in
|===

=== Schedule Risks

[cols="2,1,2"]
|===
| Risk | Severity | Mitigation

| Haskell verification delays
| Medium
| Can ship v0.2.0 without Liquid

| Learning pipeline scope creep
| High
| MVP approach, iterate

| External security audit
| Medium
| Schedule early, parallelize
|===
