= test/ — ExUnit Test Suite

Hypatia's ExUnit tests. Run with `mix test`.

== Tagging

Tests that require the `verisim-data` git-backed flat-file store are tagged
`@moduletag :verisim_data` or `@describetag :verisim_data` and are excluded
from local runs by default (see `test_helper.exs`).

Run the full suite including verisim-data tests:

[source,bash]
----
mix test --include verisim_data
----

== Subdirectories

* `rules/` — Tests for the `lib/rules/` rule engine modules
