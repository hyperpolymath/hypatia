= Hypatia: Quickstart Guide
:toc: macro
:icons: font
:source-highlighter: rouge

Neurosymbolic CI/CD intelligence layer. Coordinates gitbot-fleet via a safety
triangle pipeline with 5 neural networks, VQL queries, Bayesian confidence, and
Logtalk pattern detection.

toc::[]

== Prerequisites

[cols="1,1,2"]
|===
| Tool | Version | Install

| Elixir
| >= 1.14
| `asdf install elixir latest` or https://elixir-lang.org/install.html

| Erlang/OTP
| >= 25
| `asdf install erlang latest`

| Rust
| nightly (>= 1.80)
| `asdf install rust nightly` or https://rustup.rs

| just
| >= 1.0
| `cargo install just` or https://just.systems

| pkg-config + openssl-devel
| system
| `dnf install pkg-config openssl-devel` / `apt install pkg-config libssl-dev`
|===

Optional (for ABI/FFI layers):

* Idris2 >= 0.7.0 (for ABI definitions in `src/abi/`)
* Zig >= 0.13.0 (for FFI bridge in `ffi/zig/`)

== Clone and build

[source,bash]
----
cd ~/Documents/hyperpolymath-repos
git clone https://github.com/hyperpolymath/hypatia.git
cd hypatia

# Build everything (Rust workspace + Elixir)
just build-all
----

Expected output:

----
   Compiling hypatia-cli v0.1.0
   Compiling hypatia-data v0.1.0
   ...
    Finished `release` profile [optimized] target(s) in 30s
Resolving Hex packages...
...
Compiling 15 files (.ex)
Generated hypatia app
----

== Run a scan

[source,bash]
----
# Scan a local repository
just scan /path/to/some-repo

# Or via cargo directly
cargo run --bin hyper -- scan ../proven --verbose
----

Expected:

----
[hypatia] Scanning ../proven ...
  Found 3 weak points (0 critical, 1 high, 2 medium)
  Patterns matched: PA003, PA007, PA012
  Triangle routing: 1 eliminate, 1 substitute, 1 control
----

== Build the ABI/FFI layers

[source,bash]
----
# Compile Idris2 ABI type definitions
just compile-abi

# Build Zig FFI bridge (7 exported C functions)
just build-ffi
----

== Troubleshooting

`openssl-sys build failure`::
  Install `openssl-devel` (Fedora) or `libssl-dev` (Debian/Ubuntu).

`mix: command not found`::
  Install Elixir via asdf: `asdf plugin add elixir && asdf install elixir latest`.

`** (Mix) Could not find Hex`::
  Run `mix local.hex --force && mix local.rebar --force`.

Elixir tests hang::
  The OTP supervision tree starts GenServers. Set `MIX_ENV=test` and ensure
  no other Hypatia instance is running.

`idris2 not found` (for ABI work)::
  Install via pack: https://github.com/stefan-hoeck/idris2-pack
